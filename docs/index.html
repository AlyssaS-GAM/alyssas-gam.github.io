Projects

Galmi

Idle Clicker · Browser Game

An incremental resource-management game built in JavaScript, HTML, and CSS. Players mine rocks and work through a multi-step crafting chain, unlocking new resources, workers, and upgrades along the way.

My Role Part of a small team at Winfield Studios. Contributed to frontend implementation, UI layout, and collaborated on systems design including the resource crafting chain.
Tools HTML · CSS · JavaScript
Goals & Intent The goal was to publish a real, accessible game that anyone could pick up and play. Making it browser-based was a deliberate choice to remove every possible barrier between the player and the game.
Process The resource chain was designed before it was coded, with an initial set of dependencies and production rates mapped out as a starting point. As the project grew, a few additional resources were added beyond what was originally planned. Worker and housing systems were layered in to give players control over their production speed. The CSS background was built iteratively, testing across screen sizes to ensure it held up without any image files.
Technical Focus Multi-step resource crafting chain, worker and housing systems that interact with production rates, and a game background built entirely in CSS with no image assets, scaling responsively across all screen sizes.
What I Learned Working on Galmi deepened my understanding of how resource systems need to be balanced; too fast and there's no tension, too slow and players disengage.
