Projects

Galmi

Idle Clicker · Browser Game

An incremental resource-management game built in vanilla JavaScript. Players mine rocks and work through a multi-step crafting chain, smelting ore into iron, blasting into steel, reducing chromite into ferrochrome, and unlocking new resources, workers, and upgrades along the way.

My Role Part of a small team at Winfield Studios. Contributed to frontend implementation, UI layout, and systems design, including portions of the resource crafting chain logic.
Tools HTML · CSS · Vanilla JavaScript
Technical Focus Multi-step crafting chain (rocks → ore → iron → steel → ferrochrome), worker and housing systems that interact with production rates, and a game background built entirely in CSS, no image assets, scaling responsively across all screen sizes.
What I Learned Working on Galmi deepened my understanding of how resource systems need to be balanced, too fast and there's no tension, too slow and players disengage. I also discovered how much CSS alone can accomplish when used intentionally, which changed how I think about frontend work.
